# Wooting Profile
This Wooting TWO HE profile is tuned for fast, controlled play: light actuation for movement keys gives quick starts and strafes, while slightly deeper actuation on utility keys helps prevent accidental presses. Combined with Rapid Trigger behavior, the keyboard feels responsive when tapping for precise movement yet remains stable for typing and general use, making it a practical daily profile for both gaming sessions and normal desktop work.
