# til

A repository of TILs (Today I Learned): small snippets, learnings, tips, and plans.

This is primarily a markdown content repository. Each TIL is a single markdown file inside a category directory.

Initial categories:

- `.net`
- `AWX`
- `SQL`
- `PowerShell`
- `Entra`

Inspiration:

- https://til.simonwillison.net/
- https://github.com/jbranchaud/til

- https://github.com/jwworth/til
